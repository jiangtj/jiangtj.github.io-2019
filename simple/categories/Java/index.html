简介

Eureka Server是高可用的（可同时作为客户端向其他注册中心注册）服务发现的注册中心，为每个客户端（Eureka Client）提供注册服务，并提供已注册服务信息
Eureka Client向服务注册中心注册，并提供断路、负载均衡等功能
