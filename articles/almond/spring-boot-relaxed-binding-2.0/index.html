读取配置属性

如果，你需要从环境中读出属性值到你的app中，你需要使用与配置文件一样规则的名称

  • 相同的名称
  • 必须是 (a-z,0-9)字母
  • 必须是 小写
  • 唯一允许的特殊字符是[]，用于表示list的索引
  • 用连接符-分割单词
  • 不能以数字开头
